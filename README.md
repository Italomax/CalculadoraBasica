# CalculadoraBasica

# Tecnologias utilizadas
## Back end
- Cobol

First, download the `open-cobol`:
```bash
brew install open-cobol
```

Run compiler:
```bash
cobc -free -x -o calculator calculator.cblD
```

Open the program:
```bash
./calculator
```
